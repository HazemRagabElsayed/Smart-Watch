<h2 align="left">
Description:
</h2>

in this project:

-you can control the postition of Time and Date on OLED Screen by pressing buttons

-you can change the time and date that is displayed in OLED screen by pressing certain buttons

-you can also change the unit of temperature which is displayed in OLED screen through buttons


###

<h2 align="left">
Devices used:
</h2>

-Atmega32 (where we put our code to do specific functions)

-OLED 128 x 64 (for displaying Date, Time and Temperature)

-buttons (for controlling different things)

<h2 align="left">
Tools used:
</h2>

-Proteus 8 Professional

-Atmel Studio 6.0

<h2 align="left">
note:
</h2>

-Data memory (SRAM) of Atmega32 is very limited so I stored arrays on Program Memory (FLASH)

-bugs are tested so user cannot enter invalid Date or Time also user can't make the position of time and date outside the screen
